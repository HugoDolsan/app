# Planejamento HD

App de planejamento de tarefas (PWA) com calendário mensal e tabela editável, sincronizado sob demanda com a planilha Google **Planejamento_Tarefas_HD1_2025** via Apps Script.

- Site: https://planejamentotarefas-hd.vercel.app
- Instruções completas: [LEIA-ME.md](LEIA-ME.md)
- Cada push na branch `main` publica automaticamente no Vercel.
